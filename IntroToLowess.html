Introduction to Locally Weighted Regression

Main Idea: Trends and associations are generally nonlinear; however, locally, trends can be interpreted linearly.

Kernel function: we use a bell shaped kernel function with parameter tau to solve local linear regression.

  • tau refers to the bandwidth of the distribution
  • Larger tau will result in a smoother curve.

Framework based on linear regression

The main idea of linear regression is the assumption that:
y=X⋅β+σϵy = X\cdot\beta +\sigma\epsilony=X⋅β+σϵ

If we pre-multiply this equation with a matrix of weights (the “weights” of observation iii are on the main diagonal and the rest of the elements are 0) we get:

The independent observations are the rows of the matrix XXX. Each row has a number of columns (this is the number of features) and we can denote it by p.p.p. As such, every row is a vector in Rp.\mathbb{R}^p.Rp.

We shall have nnn different weight vectors because we have nnn different observations.

Important aspect: linear regression can be seen as a linear combination of the observed outputs (values of the dependent variable).

We have:

XTy=XTXβ+σXTϵX^Ty = X^TX\beta +\sigma X^T\epsilonXTy=XTXβ+σXTϵ

We solve for β\betaβ (by assuming that XTXX^TXXTX is invertible):

β=(XTX)−1(XTy)−σ(XTX)−1XTϵ\beta = (X^TX)^{-1}(X^Ty) - \sigma (X^TX)^{-1}X^T\epsilonβ=(XTX)−1(XTy)−σ(XTX)−1XTϵ

We take the expected value of this equation and obtain the expected value of the coeffcients:

βˉ=(XTX)−1(XTy)\bar{\beta} = (X^TX)^{-1}(X^Ty)βˉ​=(XTX)−1(XTy)

Therefore the predictions we make are:

y^=X(XTX)−1(XTy) \hat{y} = X(X^TX)^{-1}(X^Ty)y^​=X(XTX)−1(XTy)

For the locally weighted regression we have

For locally weighted regression, y^\hat{y}y^​ is obtained as a different linear combination of the values of y.

Python Implementation:

def  lowess_bell_shape_kern(x, y, tau = .005):
	n = len(x)
	yest = np.zeros(n)
	
	# Initializing all weights from the bell shape kernel function
	# here w is an nxn matrix
	w = np.array([np.exp(- (x - x[i])**2/(2*tau)) for i in  range(n)])
	
	#Looping through all x-points
	for i in  range(n):
		weights = w[:, i]
		b = np.array([np.sum(weights * y), np.sum(weights * y * x)])
		A = np.array([[np.sum(weights), np.sum(weights * x)], [np.sum(weights * x), np.sum(weights * x * x)]])
		theta = linalg.solve(A, b)
		yest[i] = theta[0] + theta[1] * x[i]
		
	return yest
